# 💻 **FTT Blazor**
> Estudo funcional do Blazor comparando com o Angular e entrega de uma biblioteca com base em Material Design

---

## ☕ **FTTBlazor Beta**
> Componentes desenvolvidos até agora
- Dialog;
- Modal;
- Toast;
- Forms:
  - Input;
  - Number;
  - Color;
  - TextArea;
  - Calendar;
  - Time;
  - CheckBox;
  - Telefone;
  - EMail;
  - Link;
  - Select:
    - Select Type T com endpoint;
    - Select Enum;
    - Select Array de string;
